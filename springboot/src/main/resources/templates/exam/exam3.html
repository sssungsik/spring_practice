thymeleaf 조건문

- 태그 속성 th:if, th:unless, th:switch
- th:if -> 조건식이 true일 때 화면에 보이는 조건문
- th:unless -> 조건식이 false일 때 화면에 보이는 조건문
- th:switch -> 값에 일치하는 요소만 화면에 보이는 조건문
- 유효범위 : 선언한 태그가 닫힐때까지

true일 경우 화면 출력(if)

false일 경우 화면 출력(else)

값과 일치된 요소만 화면 출력

값과 일치된 요소만 화면 출력

값과 일치된 요소만 화면 출력(default 구문)

thymeleaf 지역변수 선언

- th:with="변수명=${'값'}"
- th:with 선언한 요소가 존재한다면 변수의 유효범위 태그 닫힐때 까지

thymeleaf 블록요소

- th:block : thymeleaf에서 블록구문을 만들때 사용
- 화면에 렌더링되는 대상이 아니다. 프로그램언어 { }

일치하면 회원이름 출력

일치하지 않으면 회원연락처 출력

값을 참조할 수 없으므로 렌더링 x

thymeleaf 객체바인딩

- th:object 구문
- 예시: th:object="${객체}"
- 선언한 태그 자식 노드에서는 th:text="*{객체 속성명}" *로 참조가능
